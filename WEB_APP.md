# CMUdict Flask Web App

This is a minimal Flask app for searching the CMU Pronouncing Dictionary.

## Run it

1) Create and activate a virtual environment (optional but recommended).
2) Install Flask.
3) Start the app and open it in your browser.

### Windows PowerShell

```powershell
python -m venv .venv
.venv\Scripts\Activate.ps1
pip install Flask
python web_app.py
```

Open `http://127.0.0.1:5000` in your browser.

### macOS/Linux

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install Flask
python web_app.py
```

Open `http://127.0.0.1:5000` in your browser.
